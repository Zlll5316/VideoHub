# 快速创建数据库表指南

## ⚠️ 如果看到"加载团队数据失败"错误

这说明数据库表还没有创建。请按照以下步骤操作：

## 📋 步骤 1：登录 Supabase

1. 访问 https://supabase.com/dashboard
2. 登录你的账号
3. 选择你的项目

## 📋 步骤 2：打开 SQL Editor

1. 在左侧菜单中找到 "SQL Editor"
2. 点击进入

## 📋 步骤 3：执行 SQL 脚本

1. 打开项目中的 `数据库表结构.sql` 文件
2. 复制**所有内容**
3. 粘贴到 Supabase SQL Editor 中
4. 点击 "Run" 或按 `Cmd/Ctrl + Enter` 执行

## 📋 步骤 4：验证表已创建

执行完成后，你应该看到：
- ✅ 4 个表已创建：`teams`, `team_members`, `team_folders`, `team_videos`
- ✅ 所有表的 RLS 已启用
- ✅ 索引已创建

## 📋 步骤 5：刷新页面

回到应用，刷新团队空间页面，应该就能正常加载了！

## 🔍 如果还是失败

1. **检查错误信息**：
   - 打开浏览器控制台（F12）
   - 查看具体的错误信息

2. **常见问题**：
   - **表不存在**：确保 SQL 脚本完全执行成功
   - **权限错误**：确保 RLS 策略已正确创建
   - **网络错误**：检查 Supabase 连接是否正常

3. **手动检查表**：
   - 在 Supabase Dashboard 中
   - 进入 "Table Editor"
   - 确认能看到 `teams`, `team_members`, `team_folders`, `team_videos` 四个表

## 💡 提示

- 如果 SQL 执行报错，可能是某些表已存在，可以忽略或先删除旧表
- 确保你的 Supabase 项目 URL 和 Key 在 `.env` 文件中配置正确
